# Medical Review Analysis System

A backend system for intelligent analysis of patient reviews using Natural Language Processing (NLP) and fuzzy inference. The project transforms unstructured textual feedback into interpretable quantitative healthcare quality metrics by combining linguistic feature extraction with mathematical decision-making models.

The system demonstrates the integration of classical Artificial Intelligence techniques, fuzzy logic, and NLP to support explainable healthcare service evaluation.

---

## Overview

Traditional rating systems often fail to capture the complexity of patient feedback. This project addresses this limitation by extracting semantic information from natural language reviews and modeling uncertainty through fuzzy logic.

The resulting score is derived from multiple linguistic and behavioral indicators rather than relying solely on explicit user ratings.

---

## Core Components

- Sentiment and semantic analysis of patient feedback
- Expert-driven fuzzy inference engine
- Multi-criteria decision-making model
- Relational data storage using SQLAlchemy ORM

## Healthcare Quality Indicators

The fuzzy model evaluates multiple dimensions of healthcare quality:

- Waiting time
- Doctor competence
- Staff politeness
- Cleanliness
- Consultation quality
- Emotional sentiment

Each indicator is represented as a linguistic variable and evaluated using fuzzy membership functions before contributing to the final quality score.

## Research Motivation

The project explores the application of fuzzy mathematical models for healthcare quality assessment based on natural language feedback. By combining NLP with fuzzy inference, the system produces transparent and interpretable evaluations while preserving the uncertainty inherent in human language.

This approach aligns with modern Explainable AI (XAI) principles, where model decisions remain understandable and traceable rather than being generated by a black-box predictor.
